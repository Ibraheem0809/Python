## 🔢 NumPy

### Topics Covered

* ✅ **Introduction to NumPy** – Learned the basics of numerical computing with NumPy.
* ✅ **2D & 3D Arrays and Shapes** – Worked with multi-dimensional arrays and understood their structure.
* ✅ **1D & 2D Slicing (Rows & Columns)** – Selected and accessed specific parts of arrays.
* ✅ **Arithmetic Operations** – Performed mathematical operations on arrays efficiently.
* ✅ **Broadcasting** – Applied operations between arrays of different shapes.
* ✅ **Aggregate Functions** – Calculated sums, averages, minimums, maximums, and more.
* ✅ **Boolean Filtering** – Filtered data using conditions.
* ✅ **Random Number Generation** – Generated random values for testing and simulations.
* ✅ **Shuffle & Choice** – Randomly shuffled data and selected random elements.

### Topics Covered

* ✅ **DataFrames** – Managed and analyzed tabular data using rows and columns.
* ✅ **Importing Data** – Loaded data from CSV and other file formats.
